
##### Bilibili 南小濡

## 留言板项目后台使用说明

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
node/nodemon app.js
```

# 注意事项

数据库版本必须>=5.7,否则不支持JSON字段.本项目开发时使用的版本为8.0.12;
Node.JS版本:本项目开发时使用的版本为12.14.1;
在使用之前请先将根目录下的"messageboard"数据库导入,然后在util > msg.config.js中将发件邮箱信息填入;
